# Expense Manager

A terminal-based expense tracking system built with 
Python, designed for working professionals to manage 
their monthly finances effectively.

## Features
- Add monthly salary
- Add expenses with 7 categories:
  - Food
  - Transport
  - Medical
  - Bills/Loan/EMI
  - Entertainment
  - Savings
  - Other
- View all expenses
- Monthly summary with category breakdown
- Visual spending health scale
- Data automatically saved to JSON file
- Data loads automatically on startup

## Spending Health Scale
90%+ remaining → Excellent.
60%+ remaining → Good.
40%+ remaining → Be Careful.
20%+ remaining → Warning.
10%+ remaining → Danger.

## How To Use
1. Run the program
2. Add your monthly salary first
3. Add expenses as you spend
4. Check summary to see spending breakdown
5. Check health scale to see your financial health

## Built With
- Python 3
- JSON for data storage
- datetime for timestamps

## Target Users
Working professionals who want to track 
and manage their monthly expenses
